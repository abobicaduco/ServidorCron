Open-source dashboard

The web UI shipped in this repository is dashboard.html. Run python main.py and open the URL shown in the console.

Private / enterprise UI: copy dashboard.html to ServidorCron.html (listed in .gitignore), customize branding, and point your private server file to that path. Do not commit confidential layouts.

Versão em português: o painel público é o arquivo dashboard.html. Para um painel interno, copie para ServidorCron.html (ignorado pelo Git) e ajuste o caminho no seu backend privado.
